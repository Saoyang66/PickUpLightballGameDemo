# 拾取光球游戏Demo

## 当前功能（Day01）
- 利用遥杆操控主角（一个绿色球）去拾取小球，小球分为蓝色小球（加一分），金色小球（加五分），红色小球（道具：炸弹）
- 在拥有炸弹的时候点击投掷炸弹按钮，再次点击屏幕，则会在该位置投掷出炸弹，有爆炸冲击波动画效果（具体功能未做）
- 炸弹拥有拾取上限（待完善）（已完善，见Day02）
